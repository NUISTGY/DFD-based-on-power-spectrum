# DFD-based-on-power-spectrum
基于功率谱的Deepfake Detection

## 使用了三种分类器：SVM，LR，神经网络

在训练集验证集为40k张的情况下（8：2）：
 ```
    SVM acc达到96.6%
    LR  acc达到70%
    简单神经网络在1000次epochs情况下可达85%acc
 ```
